# Prompts para Communication Plan

## Prompt para Plano de Comunicação Completo

```
Você é um especialista em comunicação e gerenciamento de projetos. Preciso da sua ajuda para criar um Plano de Comunicação completo para o projeto [nome do projeto].

Contexto do projeto:
- Descrição: [breve descrição do projeto]
- Duração: [timeline do projeto]
- Tamanho da equipe: [tamanho]
- Principais grupos de stakeholders: [listar principais grupos]
- Distribuição geográfica: [local, distribuído, internacional]
- Complexidade/sensibilidade: [baixa, média, alta]

Por favor, crie um Plano de Comunicação detalhado que inclua:

1. Análise dos stakeholders e suas necessidades de comunicação
2. Matriz de comunicação completa (tipo, frequência, método, audiência, responsável)
3. Modelos e templates para comunicações principais
4. Procedimentos de comunicação (aprovação, escalação, crise)
5. Ferramentas e tecnologias recomendadas
6. Métricas para avaliar a eficácia das comunicações

Para cada componente, forneça recomendações práticas e considere as particularidades deste projeto específico.
```

## Prompt para Análise de Stakeholders

```
Como especialista em comunicação estratégica, preciso realizar uma análise abrangente dos stakeholders para o projeto [nome do projeto] para desenvolver um plano de comunicação eficaz.

Contexto do projeto:
- Tipo de projeto: [tipo]
- Escopo: [escopo]
- Impacto organizacional: [baixo, médio, alto]
- Principais resultados esperados: [resultados]

Por favor, ajude-me a:

1. Identificar todos os grupos de stakeholders relevantes:
   - Stakeholders internos (equipe do projeto, gerência, outros departamentos)
   - Stakeholders externos (clientes, fornecedores, reguladores)
   - Stakeholders primários e secundários

2. Para cada grupo identificado, analisar:
   - Nível de influência (baixo, médio, alto)
   - Nível de interesse (baixo, médio, alto)
   - Atitude em relação ao projeto (apoiador, neutro, resistente)
   - Conhecimento sobre o projeto
   - Expectativas principais

3. Mapear as necessidades de comunicação de cada grupo:
   - Tipo de informação necessária
   - Nível de detalhe apropriado
   - Frequência ideal
   - Formato preferido
   - Sensibilidades específicas

4. Desenvolver estratégias de comunicação para cada categoria:
   - Abordagens para stakeholders de alta influência/alto interesse
   - Táticas para convertir resistentes em apoiadores
   - Métodos para manter stakeholders adequadamente informados
   - Considerações especiais para stakeholders-chave

Inclua um modelo visual de mapa de stakeholders e uma matriz de análise de impacto/influência com recomendações específicas para cada quadrante.
```

## Prompt para Matriz de Comunicação Detalhada

```
Como gerente de comunicações, preciso desenvolver uma matriz de comunicação detalhada para o projeto [nome do projeto].

Já identificamos os seguintes grupos de stakeholders principais:
- [Grupo de stakeholder 1]
- [Grupo de stakeholder 2]
- [Grupo de stakeholder 3]
- [Outros grupos relevantes]

Por favor, ajude-me a criar uma matriz de comunicação abrangente que inclua:

1. Para cada tipo de comunicação, definir:
   - Nome/título da comunicação
   - Objetivo/propósito
   - Conteúdo principal
   - Formato (relatório, e-mail, apresentação, reunião, dashboard)
   - Nível de formalidade (formal, semi-formal, informal)
   - Frequência (diária, semanal, quinzenal, mensal, por evento)
   - Audiência (quais stakeholders)
   - Método de entrega (e-mail, plataforma, reunião presencial, videoconferência)
   - Responsável pela produção
   - Responsável pela distribuição/apresentação
   - Tempo de antecedência necessário
   - Feedback esperado (sim/não, tipo de feedback)
   - Critérios de sucesso

2. Categorizar as comunicações por:
   - Comunicações de rotina
   - Comunicações de progresso
   - Comunicações de mudança
   - Comunicações de risco/problema
   - Comunicações de decisão
   - Comunicações de marco/milestone

3. Incluir detalhes especiais para:
   - Comunicações para a alta administração
   - Comunicações para equipes técnicas
   - Comunicações para clientes e usuários finais
   - Comunicações regulatórias ou de conformidade

Forneça a matriz em formato tabular bem estruturado, com exemplos específicos para este projeto e notas explicativas quando necessário.
```

## Prompt para Procedimentos de Comunicação

```
Como especialista em governança de comunicações, preciso estabelecer procedimentos claros de comunicação para o projeto [nome do projeto].

Contexto do projeto:
- Sensibilidade das informações: [baixa, média, alta]
- Requisitos regulatórios: [se aplicável]
- Cultura organizacional: [descrição relevante]
- Desafios de comunicação conhecidos: [desafios específicos]

Por favor, ajude-me a desenvolver:

1. Procedimentos de aprovação para comunicações externas:
   - Níveis de aprovação por tipo de comunicação
   - Fluxo de aprovação detalhado
   - Prazos para aprovação
   - Processo para aprovações urgentes
   - Documentação de aprovações

2. Protocolo para comunicações de crise:
   - Definição de situações de crise
   - Equipe de comunicação de crise
   - Sequência de notificações
   - Mensagens pré-aprovadas para cenários comuns
   - Porta-vozes designados
   - Canais de comunicação de emergência

3. Processo para escalação de problemas:
   - Critérios para escalação
   - Matriz de escalação (quem contactar para cada tipo de problema)
   - Informações necessárias ao escalar
   - Prazos de resposta esperados
   - Acompanhamento e fechamento

4. Diretrizes para comunicações confidenciais:
   - Classificação de informações
   - Controles de acesso
   - Métodos seguros de transmissão
   - Requisitos de retenção/exclusão
   - Tratamento de violações de confidencialidade

5. Processo para gestão de feedback:
   - Métodos de coleta de feedback
   - Análise e triagem de feedback
   - Responsabilidades de resposta
   - Incorporação ao ciclo de comunicação

Inclua fluxogramas detalhados para cada procedimento e modelos/templates para documentação associada.
```

## Prompt para Métricas e Avaliação de Comunicações

```
Como analista de eficácia de comunicações, preciso desenvolver um sistema de métricas e avaliação para o plano de comunicação do projeto [nome do projeto].

Contexto:
- Objetivos principais do projeto: [objetivos]
- Desafios específicos de comunicação: [desafios]
- Grupos-chave de stakeholders: [grupos principais]
- Canais de comunicação utilizados: [canais]

Por favor, ajude-me a criar:

1. Indicadores-chave de desempenho (KPIs) para comunicações:
   - KPIs quantitativos (taxas de abertura, participação, tempo de resposta, etc.)
   - KPIs qualitativos (clareza percebida, utilidade, satisfação, etc.)
   - Indicadores de engajamento
   - Métricas de alinhamento com objetivos do projeto

2. Métodos de coleta de dados:
   - Ferramentas automáticas de rastreamento
   - Pesquisas e questionários
   - Entrevistas e grupos focais
   - Observação direta
   - Análise de feedback espontâneo

3. Processo de análise e reporting:
   - Frequência de avaliação
   - Formato de relatórios
   - Responsáveis pela análise
   - Distribuição dos resultados
   - Integração com outros relatórios do projeto

4. Ciclo de melhoria contínua:
   - Processo para identificar áreas problemáticas
   - Método para desenvolver ações corretivas
   - Implementação de melhorias
   - Reavaliação após mudanças
   - Documentação de lições aprendidas

5. Benchmarks e padrões:
   - Referências internas
   - Melhores práticas da indústria
   - Metas específicas por tipo de comunicação
   - Valores limites para ações corretivas

Forneça exemplos de dashboards de comunicação, templates de pesquisa, e planos de ação para cenários comuns de problemas de comunicação identificados através das métricas.
```
