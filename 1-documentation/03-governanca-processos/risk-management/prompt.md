# Prompts para Risk Management

## Prompt para Estrutura Completa de Gestão de Riscos

```
Você é um especialista em gestão de riscos de projetos. Preciso criar um documento de Risk Management para o projeto [nome do projeto].

Contexto:
- Descrição do projeto: [breve descrição]
- Equipe: [tamanho e composição]
- Duração: [cronograma]
- Principais entregáveis: [lista]

Por favor, elabore um documento que inclua:
1. Metodologia de gestão de riscos (abordagem, ferramentas, critérios)
2. Processo de identificação e registro de riscos
3. Análise qualitativa e quantitativa dos riscos
4. Planejamento e implementação de respostas
5. Monitoramento, controle e comunicação de riscos
6. Papéis e responsabilidades
7. Templates de registro de riscos

Inclua exemplos práticos e recomendações para cada seção.
```

## Prompt para Identificação e Registro de Riscos

```
Como gerente de projeto, preciso identificar e registrar riscos para o projeto [nome do projeto].

Contexto:
- Fase do projeto: [fase atual]
- Principais atividades: [lista]
- Restrições conhecidas: [restrições]

Liste os riscos potenciais, descreva cada um, classifique por categoria (técnico, externo, organizacional, etc.) e sugira um template para registro.
```

## Prompt para Análise de Riscos

```
Como analista de riscos, preciso realizar análise qualitativa e quantitativa dos riscos identificados no projeto [nome do projeto].

Para cada risco:
- Avalie probabilidade e impacto (use escala de 1 a 5)
- Classifique o nível de risco (baixo, médio, alto)
- Sugira critérios para priorização
- Indique riscos críticos que exigem resposta imediata

Inclua exemplos de análise para 2-3 riscos típicos do projeto.
```

## Prompt para Planejamento de Respostas a Riscos

```
Como responsável pela gestão de riscos, preciso planejar respostas para os principais riscos do projeto [nome do projeto].

Para cada risco classificado como médio ou alto:
- Defina a estratégia de resposta (mitigar, transferir, aceitar, evitar)
- Descreva as ações a serem tomadas
- Indique responsáveis e prazos
- Sugira indicadores para monitoramento da resposta

Inclua exemplos práticos de respostas para riscos comuns.
```

## Prompt para Monitoramento e Comunicação de Riscos

```
Como gerente de projeto, preciso definir como os riscos serão monitorados e comunicados durante o projeto [nome do projeto].

- Estabeleça a frequência de revisão dos riscos
- Defina como as atualizações serão comunicadas à equipe e stakeholders
- Sugira formatos de relatório e dashboards
- Indique como novas ameaças serão incorporadas ao registro

Inclua um exemplo de fluxo de comunicação de riscos.
```
